== Introduction

The Morpheus Plugin API is a Java 8 based library for creating Plugins that add functionality to Morpheus. The Plugin api supports implementing providers to Morpheus of the following types:

* UI Extensions
* Task Types
* IPAM
* DNS
* Approvals
* Cypher Modules
* Custom Reports
* Cloud Providers
* Network Providers
